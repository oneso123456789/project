모듈 설치법

1. Eclipse에서 Node.js Project 생성한다.

   File - New - Node.js Project 선택

   Project name : nodepro

   Template to use : Hello World     .... finish



2. mysql 모듈 설치
  
   nodepro 프로젝트 선택후 오른 마우스 클릭  

   StartExplorer 메뉴 선택

   Start Shell Here 메뉴 선택

   d:\workspace\nodepro> npm  install  mysql

   
   mysql 모듈 설치후 nodepro 프로젝트를 선택한 상태에서 F5키 눌러서
   새로고침을 하면 nodepro 프로젝트 하위에 node_modules 폴더가 나타남.


   다른 모듈도 위와 동일한 방법으로 설치하면됨.

   d:\workspace\nodepro> npm  install mysql
   d:\workspace\nodepro> npm  install mongojs
   d:\workspace\nodepro> npm install express 
   d:\workspace\nodepro> npm install ejs 
   d:\workspace\nodepro> npm install body-parser
